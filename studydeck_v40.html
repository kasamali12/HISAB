📋 General Instructions
1.This is a Computer Based Test (CBT). All questions are of MCQ type with single correct answer.
2.Marking Scheme: Each correct answer carries +4 marks. Each wrong answer carries −1 mark. Un-attempted questions carry 0 marks.
3.You can navigate to any question at any time using the Question Palette (📋 button) during the exam.
4.You can change your answer at any time before final submission by clicking a different option.
5.Use "Mark for Review" (🔖) to bookmark questions for later — your answer (if any) will still be saved and counted.
6.The timer will count down from the allotted time. The exam will auto-submit when time expires.
7.Once you click "Submit Exam", the test cannot be resumed.
8.Results will include subject-wise analysis, rank estimate, and comparison with NEET PG 2024 cutoffs.
